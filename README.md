# 🧠 HelpHub AI — Community Support Platform

> **SMIT Grand Coding Night 2026** — AI-powered community platform connecting students who need help with those who can provide it.

## ✨ Features

| Category | Features |
|----------|----------|
| **🏠 Landing** | Hero section, community stats, featured requests, platform overview |
| **🔐 Auth** | Login/Signup with role selection (Need Help, Can Help, Both) |
| **👤 Onboarding** | AI-powered skill & interest suggestions |
| **📊 Dashboard** | Trust score, stats cards, recent requests, AI insights |
| **🔍 Explore** | Filterable request feed by category, urgency, status |
| **📝 Create Request** | AI auto-categorization, urgency detection, tag suggestions, rewrite |
| **📄 Request Detail** | Full description, AI summary, helper list, actions |
| **💬 Messages** | Two-pane chat interface for helper-requester communication |
| **🏆 Leaderboard** | Rankings with trust scores, badges, solved count |
| **🤖 AI Center** | Trend pulse, urgency watch, mentor pool, recommendations |
| **🔔 Notifications** | Filterable notification feed with read/unread tracking |
| **👤 Profile** | Public profile with skills, trust score, edit functionality |
| **⚙️ Admin** | Moderation, user management, platform analytics |

## 🛠 Tech Stack

- **Frontend:** React 19, Vite 8, React Router 7
- **Backend:** Express, MongoDB (in-memory via mongodb-memory-server)
- **AI Engine:** Custom keyword-based intelligence service
- **Auth:** JWT + bcrypt
- **Design:** Custom CSS design system (warm beige + dark teal)

## 🚀 Quick Start

```bash
# Backend
cd server && npm install && npm start

# Frontend (new terminal)
cd client && npm install && npm run dev
```
